# 📊 Supply Chain Performance Dashboard (Power BI)

## 📌 Project Overview
This project analyzes supply chain operations including revenue, cost, transportation modes, and supplier performance using Power BI.

The dashboard provides actionable insights into logistics efficiency, cost optimization, and product distribution.

---

## 🎯 Business Problem
Companies often struggle to track:
- Transportation costs across different modes
- Supplier performance and revenue contribution
- Route optimization
- Profitability vs cost

This dashboard helps stakeholders make data-driven logistics decisions.

---

## 📊 Key Metrics
- 💰 Total Revenue: $288.15K
- 💸 Total Cost: $25.58K
- 📦 Total Products Sold: 20K

---

## 📈 Dashboard Insights

### 1. Transportation Analysis
- Road transport generates highest revenue (~$95K)
- Sea transport has lowest cost but slower delivery

### 2. Route Optimization
- Route A contributes highest cost
- Route B & C show potential for cost balancing

### 3. Supplier Performance
- Supplier 1 & 2 generate highest revenue
- Supplier 3 underperforming → improvement opportunity

### 4. Gender-Based Revenue
- Female customers contribute ~56%
- Male customers contribute ~44%
